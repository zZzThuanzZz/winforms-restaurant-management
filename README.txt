# Restaurant Management System

## 📌 Description
A desktop application for managing restaurant operations such as menu items and orders, built using C# WinForms.

## 🎯 Purpose
This project was created to practice building a complete desktop management system with real-world business logic.

## 🛠 Tech Stack
- C#
- .NET WinForms
- Visual Studio

## ✨ Features
- Menu management
- Order handling
- Basic sales workflow
- Event-driven desktop UI

## 📂 Application Structure
- Multiple forms for different restaurant operations
- Clear event handling logic
- Modular form design

## 📈 What I Learned
- Designing desktop management systems
- Handling complex user interactions in WinForms
- Applying business logic to real-world scenarios

## ⚠️ Note
This project focuses on desktop application logic and UI design rather than cloud-based backend integration.
